=========
Les \vars
=========

Pour \amo, une \var est un \obj associé à un type "simple" via le mot-clé amo::''var'' comme dans l'exemple suivant (noter au passage l'usage possible de commentaires à la sauce \code_c).

amo::
	/* ------------------- *
	 * -- MES VARIABLES -- *
	 * ------------------- */

	var a : \R    // Une variable réelle.
	var k : \N    // Une variable naturelle.


La syntaxe générale est de la forme amo::''var nom_var : TYPE''. Dans cette section, nous nous intéressons juste aux types "simples" fournis par défaut.


note::
	La section cf::''new-func'' présente les types "composés", et la section cf::''new-type'' explique comment ajouter de nouveaux types.
